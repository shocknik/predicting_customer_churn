# Прогнозирование оттока клиентов

## Описание
Проект посвящён построению модели прогнозирования оттока клиентов.
Используется логистическая регрессия с пайплайном предобработки.

## Стек
- Python
- pandas
- scikit-learn
- joblib

## Содержимое репозитория
- `Модель прогнозирования оттока клиентов.ipynb` — основной ноутбук с анализом и обучением
- `final_pipeline_model.joblib` — сохранённый пайплайн модели
- `requirements.txt` — зависимости проекта

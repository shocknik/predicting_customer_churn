# Прогнозирование оттока клиентов

## Описание
Проект посвящён построению модели прогнозирования оттока клиентов.
Используется логистическая регрессия с пайплайном предобработки.

**Основная бизнес-метрика:** PR-AUC (Average Precision), так как класс сильно несбалансирован (~6% оттока).

**Итоговые метрики на тестовой выборке:**
- PR-AUC: **0.705**
- Precision: **0.846**
- Recall: **0.437**
- F1-score: **0.576**
